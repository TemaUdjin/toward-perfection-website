# Решения и причины

## 2026-04-22

### Продукт — личный бренд, не платформа
Юджин строит под себя, не под других тренеров. Причина: фокус, скорость запуска, личное доверие аудитории.

### Instagram DM анализ — не основной канал
Instagram API ограничивает работу с видео. Используем как лид-магнит (один бесплатный анализ), а не как основной продукт.

### Telegram бот — не подходит
Слишком неудобен как пользовательский продукт. Только для внутреннего использования (instagram-bot проект).

### Видео тренера — ключевое отличие от конкурентов
Ни одно существующее приложение не даёт персональные видео конкретного тренера. Это главное конкурентное преимущество.
