# 2026-04-26 — Design refinement: типографика и язык дашборда

Цель: причесать визуальный язык сайта. Привести дашборд к тому же линейному коду, что лендинг и курс. Уточнить типографику hero и крупных заголовков — без введения второго шрифта.

## Контекст

До правок:
- Лендинг и курс жили на тонких линиях (`border-t`, `gap-px`), дашборд — на `rounded-2xl` карточках. Рассинхрон языка.
- Hero H1 был на `clamp(3rem, 10vw, 8rem)` + `tracking-tight` + `leading-[0.95]` — на больших экранах ощущался перекачанным.
- Был один эксперимент с serif (Fraunces) + курсивом для акцентных слов — отвергнут как декоративный шум, не соответствующий строгому голосу проекта.

## Фазы

- [x] Фаза 1. Эксперимент: добавить Fraunces (serif) для крупных заголовков + курсив для акцентных слов (`Perfection`, `when ready`). Цель — проверить, нужен ли второй шрифт.
- [x] Фаза 2. Переписать дашборд в линейный язык: убрать `rounded-2xl`, заменить карточки на `border-t` секции и `gap-px` grid для статистики. Список уровней — построчно через `border-b`. AI Form Analysis — без dashed-карточки, обычная секция с CTA в стиле "Start Practice".
- [x] Фаза 3. Откатить serif/курсив. Решение: один шрифт (Geist), один голос — соответствует строгому минимализму, заявленному в `.business/assets/brand-guidelines.md`. Удалить Fraunces из `layout.tsx` и токены `--font-serif` / класс `.font-serif` из `globals.css`.
- [x] Фаза 4. Подкрутить размеры и плотность типографики:
  - Hero H1: `clamp(2.75rem, 8.5vw, 6.75rem)`, `tracking-[-0.03em]`, `leading-[0.92]`.
  - Experience pull-quote: уменьшить с `text-2xl/3xl` до `text-xl/2xl`, чтобы не спорил с H1.
  - Final CTA H2: `clamp(2rem, 5.5vw, 4.5rem)` — чуть меньше hero для иерархии.
  - Course H1, Dashboard H1: `text-4xl/5xl font-bold tracking-[-0.03em] leading-[0.95]` — единый паттерн крупных заголовков.
- [x] Фаза 5. Линейная структура дашборда сохранена даже после отката шрифта — это решение про архитектуру, не про стиль.
- [x] Фаза 6. Smoke-test всех страниц через dev-сервер: `/`, `/dashboard`, `/course/foundation` → 200.

## Итог

Реализовано целиком. Что сделано:
- Geist остаётся единым шрифтом проекта.
- Hero, Experience и Final CTA на лендинге уплотнены — те же приёмы, без перекачки.
- Дашборд переписан в линейный язык: `border-t/border-b` секции, `gap-px` grid статистики, список уровней без карточек, AI Form Analysis встроен в общий ритм.
- Все CTA унифицированы: текст + круглая обводка со стрелкой (паттерн `Start Practice`).
- Удалены: импорт Fraunces, токен `--font-serif`, класс `.font-serif`.

Ничего не осталось, не заблокировано.
