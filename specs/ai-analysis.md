# Спецификация — AI анализ формы

## Как работает
1. Пользователь загружает видео (10–60 сек)
2. Извлекаем 3–5 ключевых кадров
3. Claude Vision анализирует форму
4. Возвращаем структурированный фидбэк

## Что анализируем
| Элемент | Критерии |
|---|---|
| Запястья | угол, нейтральность, распределение веса |
| Плечи | elevation, блок открыт/закрыт, уши между руками |
| Линия тела | hollow vs banana, прямая от кистей до носков |
| Бёдра | нейтральный таз vs anterior tilt |
| Голова | нейтральная / смотрит вперёд |
| Пальцы | grip, давление на подушечки |

## Формат фидбэка
```
✅ Что хорошо: ...
❌ Главная проблема: ...
📋 Делай это: [3 конкретных упражнения]
📅 Следующий чекин: через N недель
```

## Стиль ответа
Фидбэк пишется от лица Юджина — тепло, конкретно, без воды.
Промпт настраивается под его стиль речи.
